# MultipleAndUnitTest

This is a small python application that will print "fizz" in case a natural
number is multiple of 5, "buzz" in case it is multiple of 7 or, if multiple of
both at the same time, it will print both messages together, in the same string,
as "fizzbuzz". If the number is not multiple of neither 7 or 5, it will print
"miss" instead.

The user should run the app.py application.

PT_BR

Esse é um pequeno programa que mostra a mensagem "fizz" caso o número inserido
pelo usuário seja um número natural múltiplo de 5, e "buzz" caso ele seja múlti-
plo de 7. No caso de ser múltiplo dos 2, ambas as mensagens serão impressas jun-
tas numa string "fizzbuzz". Se o número não for múltiplo nem de 7 e nem de 5, a
mensagem impressa será "miss".

O usuário deve rodar o arquivo principal app.py para que a aplicação funcione.
